Key Finding

Claude achieves ~64% theme-level accuracy cold, dropping to ~48% at the subtheme level. Few-shot examples (+1 example per subtheme) improve accuracy by ~3-4 points — a real but modest lift. The systematic failure on Institutional Logic (0% accuracy) reveals a core limitation: Claude codes by surface topic, while expert qualitative coding requires reading the structural or psychological pattern a quote instantiates — a distinction that depends on context no single quote can supply.
